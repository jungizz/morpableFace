## 3D로 만나는 이미지 컨설팅
본 프로젝트는 사용자의 스마트폰 사진에서 피부톤과 얼굴형
을 측정하여 3D 모델로 구현한 뒤, 이를 바탕으로 퍼스널컬러
와 이미지 컨설팅을 제공하는 서비스를 개발하는 것을 목표로 
한다. 이 서비스는 컴퓨터 비전과 AI, 3D 렌더링 기술을 활용
하여 비용 효율적이고 신뢰할 수 있는 컨설팅 경험을 제공한다.
